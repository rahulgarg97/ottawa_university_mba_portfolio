# Rahul MBA Portfolio Website

This folder contains a complete static MBA portfolio website.

## Files
- `index.html` - home page and program learning outcomes self-assessment
- `bus7000.html` through `bus8500.html` - artifact pages for each MBA core course
- `assets/` - embedded artifact documents and supporting assessment documents
- `styles.css` - site styling

## How to publish quickly

### Option 1: Netlify Drop (fastest)
1. Go to Netlify Drop.
2. Drag the entire unzipped `mba_portfolio_site` folder into the upload area.
3. Copy the public website URL Netlify provides.
4. Paste that URL into the MBA Program Assessment Essay where it says `Portfolio Website:`.

### Option 2: GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Enable GitHub Pages from repository settings.
4. Use the published URL in the MBA Program Assessment Essay.

### Option 3: Google Sites
Google Sites does not directly host a static HTML folder. Use the text and descriptions from this site to build your Google Site pages manually, then embed the PDF artifacts from Google Drive.

Before submitting, test the public link in a private/incognito browser to make sure it opens without an access code.
